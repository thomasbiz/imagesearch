FreeCodeCamp API Basejump: Imagesearch micro service

User stories:

  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • I can get a list of the most recently submitted search strings.

Example usage:

https://thomasb-imagesarch.herokuapp.com/api/imagesarch/lol%20cats
https://thomasb-imagesarch.herokuapp.com/api/imagesarch/lol%20cats?offset=2

Example output

[{"url":"http://imgur.com/a/gr9LO","snippet":"Sarcastic, grumpy, lol cats","context":"https://imgur.com/user/DRSKILLLLZ"},{"url":"http://imgur.com/a/DRZaV","snippet":"My cats a modern family fan lol.","context":"https://imgur.com/user/crazycolaist"}, ...]

Example usage:

https://thomasb-imagesarch.herokuapp.com/api/latest/imagesarch

Example output

[{"term":"lol cats","when":"2016-11-10T10:16:17.604Z"},{"term":"getcat","when":"2016-11-10T11:56:12.395Z"},{"term":"lol cats","when":"2016-11-10T11:56:36.830Z"}]
